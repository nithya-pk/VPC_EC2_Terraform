## VPC_EC2_Terraform

- VPC in two availabilty zones 
- Public & Private Subnet in each AZ
- Route Tables, NAT Gateway
- Security Groups for EC2 and RDS
- RDS DB in private subnets
- EC2 instance
- Wordpress installed wtih presence in both EC2 and RDS
